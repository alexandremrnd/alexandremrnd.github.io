PROJET UNIVERSITAIRE

LE BUT DU PROJET

Ce projet est le premier site web que j'ai créé. Avec quelques maladresses, j'ai réussi à orienter mon site sur un thème d'actualité dans le foot : la Champions League. Sans maîtriser totalement le CSS à cette époque, mais en reprenant les couleurs principales de cette compétition, j'ai construis un site web d'actualités sportives fonctionnel.

Matchs à venir

L'utilisateur peut visualiser les prochaines rencontres et avoir le pronostic du site.

Actualités sportives

En accédant à l'onglet "News", l'utilisateur peut alors lire des articles de presse sur le thème du site.
