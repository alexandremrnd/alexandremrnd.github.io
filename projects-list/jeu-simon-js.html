PROJET UNIVERSITAIRE

LE BUT DU PROJET

En groupe de deux étudiants, pour clôturer notre module de programmation web, nous avons recrée le célèbre jeu du Simon en utilisant seulement le langage Javascript, accompagné légèrement de code HTML et CSS.

Comment fonctionne le jeu ?

L'utilisateur dispose de 4 ronds de couleurs différentes. Lorsqu'il commence sa partie en cliquant sur "Cliquez ici", une séquence de lumières acocmpagnés de sons correpondants saffiche sur l'écran. L'utilisateur doit alors reprosuire cette séquence lorsqu'elle est terminée en cliquant sur les ronds correspondants. Il suffit donc d'aller le plus loin pour avoir un score de plus en plus important.
